# SABER咨询网站

一个专业的沙特SABER认证咨询网站，支持中英文双语，适合小白用户快速部署使用。

## 🚀 快速开始

### 1. 启动项目
```bash
npm start
```

### 2. 访问网站
- **网站首页**: http://localhost:3000
- **管理后台**: http://localhost:3000/admin
- **邮件测试**: http://localhost:3000/test

## ✨ 功能特色

- 🌐 **双语支持** - 中英文一键切换
- 📱 **响应式设计** - 适配各种设备
- 💬 **在线咨询** - 浮动聊天窗口
- 📧 **邮件通知** - 自动发送咨询邮件
- 📝 **文章管理** - 专业内容展示
- 🔒 **安全防护** - 多重安全措施
